# Healthy Meal Agent

## 1. 프로젝트 소개

**Healthy Meal Agent**는 자연어 한 마디로 건강 기준을 통과한 한식 한 끼와 레시피를 추천하는 대화형 AI 에이전트입니다.

- **한 줄 소개**: "가볍게 500kcal 이하로" 한 마디면, 건강 기준을 통과한 한 끼를 대신 찾아줍니다.
- **주요 사용자**: 건강하게 먹고 싶은 20대, 칼로리·나트륨 등 영양 조건을 신경 쓰는 직장인
- **배경**: 최근 10년간 20대의 지방 섭취·비만·고콜레스테롤혈증이 모두 악화됐습니다. 건강한 식생활 정보는 넘쳐나지만 실제 식습관 문제는 여전합니다.
- **최종 결과물**: 자연어 요청을 받아 실제 한식 한 끼 + 영양 정보 + 레시피/조리 영상을 돌려주는 웹 서비스

---

## 2. 문제 정의

> 건강한 한 끼를 원하지만, 매번 메뉴를 찾고 여러 음식의 영양을 비교·계산하는 과정이 너무 번거롭다.

- 건강하게 먹고 싶은 **의지는 있지만**, 메뉴를 고르고 각 음식의 칼로리·나트륨을 일일이 비교·계산하는 과정이 번거롭습니다.
- 기존 서비스는 단일 음식 정보만 주거나, 조건(저염·칼로리·알레르기)을 한 번에 반영해 **한 끼 전체를 구성**해 주지 못합니다.
- 건강한 식습관을 실천하지 못하는 핵심 원인이 "정보 부족"이 아니라 **"번거로움"**이라는 점을 20대 인터뷰에서 확인했습니다.

---

## 3. 문제 해결

원하는 조건을 **자연어로 입력**하면, 음식들을 조합해 **KDRI(한국인 영양소 섭취기준)로 검증한 한 끼**를 추천하고, **레시피와 조리 영상**까지 제공합니다.

**핵심 원칙 — "판단은 AI, 계산·검증·안전은 코드"**
- 조건 이해·음식 조합 등 판단은 AI(LLM)가 담당
- 칼로리·나트륨 계산과 건강 기준 검증은 코드가 담당 → AI가 영양 수치를 지어내지 않음

**전체 동작 흐름 (LangGraph 기반 에이전트)**
```
요청 이해 → 조건 추출 → 음식 검색 → 한 끼 조합 → 영양 합산 → 건강 기준 검증 → (실패 시 재시도)
```
예) "김치찌개 있는 한식 500kcal 이하로" → 완자김치찌개 + 쌀밥 (461kcal, 나트륨 358mg) → 기준 통과 → 레시피와 함께 추천

**안정성**
- 메인 AI(Upstage Solar)가 실패하면 보험 모델(OpenAI)로 자동 전환해 서비스가 끊기지 않음

---

## 4. 핵심 기능

- 자연어 조건 입력 (예: "얼큰한 국물 500kcal 이하", "계란 빼고 담백하게")
- 취향·조건을 함께 반영하는 음식 검색 (의미 검색 + 조건 필터)
- 밥·국·반찬을 조합한 한 끼 구성
- **KDRI 기준 영양 검증** (칼로리·나트륨·탄단지) 및 기준 미달 시 재추천
- **알레르기·기피 재료 자동 제외** (음식 이름뿐 아니라 들어간 재료까지 검사)
- 레시피 및 조리 영상 연계
