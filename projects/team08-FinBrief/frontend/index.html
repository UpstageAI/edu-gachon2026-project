근거 기반 리포트

모든 해석은 공신력 있는 뉴스 소스를 근거로 작성되어 신뢰할 수 있습니다.

fact_check

정확성 검증

생성된 카드의 수치를 원본 지표값과 대조하고 출처 뉴스와 교차 검증해 오류를 방지합니다.

hub

멀티채널 지원

디스코드는 물론 슬랙까지 지원하여, 사용하는 협업 채널로 바로 브리핑을 받습니다.

어떻게 동작하나요?

FinBrief는 매일 아침 LangGraph 파이프라인으로 뉴스·지표를 수집하고, 구독 토픽별로 병렬(FanOut) 처리하여 RAG 근거 위에 카드를 생성한 뒤 Discord·Slack으로 개인화 발송합니다.

  • 1 뉴스 임베딩 → Supabase pgvector 적재 & 전체 지표 리포트 생성
  • 2 토픽별 FanOut 병렬 — RAG 근거 검색 후 카드(텍스트+이미지) 생성
  • 3 Discord·Slack 개인화 발송 + Langfuse로 LLM 호출·비용 관측
